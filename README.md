# mIAu
Projeto pessoal desenvolvido durante minha transição para a área de tecnologia, com foco em aprendizado prático e evolução constante. A ideia aqui não é apresentar algo perfeito, mas sim documentar o processo de construção, erros e melhorias ao longo do caminho.

## Sobre o Projeto

Este projeto foi criado como forma de colocar em prática os primeiros conhecimentos em desenvolvimento frontend. Cada parte dele reflete uma etapa do meu aprendizado, desde a estrutura básica até tentativas de melhorar organização, design e funcionalidade.

## Objetivo

- Aprender desenvolvimento na prática  
- Entender melhor a lógica por trás do código  
- Evoluir gradualmente em HTML, CSS e JavaScript  
- Criar projetos reais, mesmo que simples  

## Tecnologias Utilizadas

- HTML  
- CSS  
- JavaScript  

## Status do Projeto

Em desenvolvimento. Este projeto pode passar por mudanças frequentes conforme eu aprendo coisas novas e aplico melhorias.

## Observações

Este repositório faz parte do meu processo de entrada na área de tecnologia. Ainda estou aprendendo, então algumas soluções podem não ser as melhores, mas fazem parte da evolução.
